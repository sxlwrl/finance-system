# Проєктування системи

Вбудовування зображень діаграм здійснюється з використанням сервісу plantuml.com (opens new window).

В markdown-файлі використовується опис діаграми

    border-radius:4px;
    border: 1px solid #cfd7e6;

>

@startuml

participant Client

participant SR as "Service Registry"

participant Service

Service -> SR : register
SR -> SR
SR --> Service
...

SR -> Service: heartbeat
SR <-- Service: health
...

Client -> SR: find
Client <-- SR: service endpoint
Client -> Service: request
Client <-- Service: response



@enduml

</center>
